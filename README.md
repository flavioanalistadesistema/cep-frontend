# Projeto: cep-frontend

## Clone the project

`git clone git@github.com:flavioanalistadesistema/cep-frontend.git`

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
